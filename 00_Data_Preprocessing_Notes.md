# Step 0: Data Preprocessing and Initialization

## 📥 1. Upstream Raw Data
The raw FASTQ files and the initial Cell Ranger output (Raw Feature-Barcode Matrices) generated by the sequencing facility are massive in size and are deposited in the **Genome Sequence Archive (GSA)** under the accession number: **CRA043742**. 

Researchers who wish to reproduce the pipeline from the absolute beginning should download the raw count matrices from GSA.

## ⚙️ 2. Standard Preprocessing Pipeline
To generate the `.rData` / `.rds` Seurat object used as the starting point for our downstream biological mining (Scripts `01` to `04`), the following standard pipeline was executed:

1. **Alignment & Quantification:** Raw reads were mapped to the maize B73 reference genome (RefGen_v5) using `Cell Ranger v6.1.2` (10x Genomics).
2. **Matrix Initialization:** The resulting filtered feature-barcode matrices were loaded into R to initialize the `Seurat` object.
3. **Basic Filtering:** Cells were filtered based on strict QC metrics:
   - `nFeature_RNA` > 500 and < 5,000
   - `nCount_RNA` > 1,000 and < 50,000
   - Organelle reads < 5%
4. **Batch Effect Correction:** `Harmony` was applied to integrate data across different biological replicates and heat stress time points (T0, T2, T24).

## 🚀 3. Downstream Analysis
Once the quality-controlled and integrated Seurat object (`maize_atlas_cca.rData`) was generated, we proceeded with the highly customized downstream multi-omics analyses, which are strictly documented in scripts `01_snRNA_seq_QC_and_Clustering.R` through `04_Spatial_ISS_Quantification.R`.
